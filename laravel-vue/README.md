## Vue.js Eğitim

### Laravel projesine Vue.js entegre etmek.
-   ```composer require laravel/ui```
-   ```php artisan ui vue```
-   ```npm install ```
-   ```npm run watch```

- **Sayfamızda Gerekli tanımlamalrını yapıyoruz**
    - ```<link rel="stylesheet" href="{{ asset('css/app.css')}}">```
    - ```<script src="{{ asset('js/app.js')}}"></script>```
